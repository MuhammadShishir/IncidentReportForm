This plugin provides an Elementor widget for creating customizable incident report forms with various functionalities.

== Description ==

Incident Report Form Elementor is a simple and easy-to-use plugin that allows you to create customized incident report forms using the Elementor page builder. With this plugin, you can easily design and deploy incident report forms on your WordPress website.

Key features include:
- Customizable form fields
- Support for uploading pictures and capturing photos
- Live location tracking of the person submitting the form
- Donation function integration
- Admin control settings for managing forms
- Built-in form builder tool for creating and customizing forms
- Additional CSS and JS settings for styling and functionality customization
- About Us page to display information about the developers
- Donation form page for accepting donations
- Shortcode generation for embedding forms into posts and pages

== Installation ==

1. Upload the `incident-report-form-elementor` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a new page or edit an existing one using the Elementor page builder.
4. Drag and drop the "Incident Report Form" widget onto your page.
5. Customize the form fields, settings, and appearance using the Elementor editor.
6. Publish or update the page to display your customized incident report form.

== Frequently Asked Questions ==

= How do I customize the form fields? =

You can customize the form fields by editing the "Incident Report Form" widget settings in the Elementor editor. Simply drag and drop the form fields you want to include and configure their settings as needed.

= Can users upload pictures and capture photos with the form? =

Yes, the plugin supports uploading pictures and capturing photos as part of the incident report form. Users can attach images to their reports to provide additional context or evidence.

= Is there a donation function available with the plugin? =

Yes, the plugin includes a donation function that allows you to accept donations directly through the incident report form. You can configure the donation settings in the plugin admin panel.
